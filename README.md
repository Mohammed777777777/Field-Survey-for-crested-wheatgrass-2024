## Overview

This repository contains the field data collection framework developed for vegetation monitoring in Saskatchewan Landing Provincial Park (SLPP), Saskatchewan, Canada.

The project includes:

- A customized ArcGIS Field Maps data collection form
- Supporting spatial datasets for site selection
- Park boundary shapefile

---

## Field Data Collection Form

The field data collection form was created using **ArcGIS Field Maps Designer** and is intended for use on mobile devices during field surveys.

The form allows standardized, spatially referenced data entry and ensures consistency across sampling locations.

### Vegetation Cover Variables Collected

- Percent cover of crested wheatgrass
- Percent cover of native grass species
- Percent cover of standing dead vegetation
- Percent cover of bare ground
- Additional ground cover observations (if applicable)

### Site-Level Information Recorded

- Elevation
- GPS coordinates (automatically captured)
- Site ID
- Date and time
- Observer information
- General site notes
